AuraGold: Multi-Frequency Predictive Modeling for Global Gold Markets

A sophisticated dual-layered analytical framework leveraging macroeconomic indicators, currency fluctuations, and technical signals to forecast daily and hourly gold price dynamics.

---
📈 Project Background
Gold remains the ultimate hedge against inflation and geopolitical instability. However, its price is influenced by a complex web of interdependencies ranging from real interest rates and currency strength to the performance of mining equities. This project was designed to move beyond simple time-series forecasting by integrating high-dimensional data—including macro indicators from FRED and real-time market data from Yahoo Finance—to build a robust prediction engine for both daily strategic and hourly tactical decision-making.

---
🛠 Technical Stack
Language: Python 3.x
Data Acquisition: yfinance (Real-time market data), pandas_datareader (Macroeconomic data via FRED).
Data Manipulation: Pandas, NumPy.
Visualization: Matplotlib, Seaborn (Correlation heatmaps, Dual Y-axis trend analysis).
Machine Learning: Scikit-learn (Linear Regression, Ridge, Lasso, ElasticNet, RFE), XGBoost.
Feature Engineering: ta (Technical Analysis library for RSI, EMA, SMA, ATR).
